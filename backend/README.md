# UNIA Chatbot Backend

Backend server for UNIA Chatbot using Express and Google Gemini API.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables in `.env`:
   ```
   GEMINI_API_KEY=your_api_key_here
   PORT=3001
   FRONTEND_URL=http://localhost:5173
   ```

3. Run the server:
   ```bash
   npm start
   # or for development
   npm run dev
   ```
